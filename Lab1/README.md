Используемые устройства: матрица светодиодов 8х8, ряд ключей 1х8
Написать ассемблерную программу, которая будет в зависимости от значения, заданного рядом ключей, зажигать светодиод на матрице светодиодов. В случае, если включается восьмой ключ, нужно отключить все светодиоды на матрице и перейти в метку end.

Данный код предназначен для запуска в эмуляторе Ripes.
В коде присутствуют комментарии к каждой строке.
